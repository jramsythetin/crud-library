# crud-library
 This is a simple CRUD project built with Java and Spring Boot.

 
## 💡 Features

- Create, Read, Update, Delete Books
- Spring Boot + JPA + Lombok
- Clean architecture (Controller, Service, Repository)
- In-memory H2 database for testing

## 🛠️ Technologies

- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Lombok
